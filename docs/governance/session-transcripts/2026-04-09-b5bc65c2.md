## Claude

**S line** — fired on every completed step:

```
S42 401 482 75
│   │   │   └─ cadence: 75 steps/min
│   │   └───── swing duration: 482 ms (foot in air)
│   └───────── stance duration: 401 ms (foot on ground)
└─────────────  step index: 42
```

**N line** — snapshot every 10 steps, rolling window average:

```
N40 62.5 75
│   │    └─ mean cadence: 75 spm
│   └────── SI stance: 62.5%  (Symmetry Index — how symmetric your gait is)
└──────────  anchor step index: 40
```

**SI stance (the key clinical number):**
```
SI = |stance_left - stance_right| / avg(stance_left, stance_right) × 100
```
- `0%` = perfectly symmetric
- `< 10%` = normal healthy gait
- `> 10%` = asymmetry worth investigating
- `> 20%` = significant impairment

**Sanity checks when you walk:**
| Signal | Expected value | Red flag |
|--------|---------------|----------|
| stance_ms | 350–700 ms | < 200 or > 900 |
| swing_ms | 300–500 ms | < 200 or > 700 |
| cadence | 80–130 spm normal walk | < 40 or > 200 |
| SI | < 10% | > 20% without known cause |
