# Crucible Amendments

> **This file is now a stub index.** The canonical source for each amendment is the
> individual file in `docs/governance/amendments/`. Enforcement checks (`article_i.py`,
> `corpus.py`, `graph.py`) read from that directory directly.
>
> For human reading, see the individual files or the Amendment Index below.
> For machine reading, use `docs/governance/amendments/MANIFEST.md`.

---

All amendments derive from Article I (Signal First) and/or Article II (Human in the Loop)
in CONSTITUTION.md. The governing Articles are unconditional and cannot be amended.

Amendment 1 is always the Domain Primitives amendment — written by `/spec collect` for
each specific project. Amendments 2–4 are mandatory framework amendments. Amendments
5 onwards are domain-agnostic operational amendments; ratify by changing status to
RATIFIED in both the individual file and `amendments/MANIFEST.md`.
